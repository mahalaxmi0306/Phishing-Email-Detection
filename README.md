# Phishing Email Detection System

A machine learning-based cybersecurity project developed using Python and Scikit-learn to detect phishing and legitimate emails.

## Features
- Detects phishing and safe emails
- URL detection
- Suspicious keyword analysis
- Urgency word detection
- Displays accuracy and confusion matrix
- Predicts unknown email messages

## Technologies Used
- Python
- Scikit-learn
- Pandas
- TF-IDF Vectorizer
